系统架构

1 文本提取层 · extractors/

  • PDFExtractor — PyMuPDF + 乱码检测 + RapidOCR + Vision API
  • DocxExtractor — python-docx 段落+表格
  • ExcelExtractor — pandas 逐Sheet逐行
  • PptxExtractor — python-pptx Slide/Shape/TextFrame
  • HTMLExtractor — BeautifulSoup 标签剥离
  • PlainTextExtractor — 编码自动检测

2 结构化解析层 · parsers/

  • LangChain Chain — ChatPromptTemplate | ChatOpenAI | JsonOutputParser
  • Prompt 模板 — prompts/llm.md 定义字段Schema
  • .env 配置 — API Key / Base URL / 模型名
  • 输出 — resumeBase / Education / Project / WorkExp

3 注册表机制

  • 策略模式 — 按扩展名分派 Extractor
  • 15 种扩展名覆盖 — .pdf .docx .xlsx .pptx .html .txt .md .csv ...
  • 零侵入扩展 — register(".xxx", MyExtractor()) 即可

4 CLI 入口 · main.py

  • python main.py file.pdf — 纯文本提取
  • python main.py file.pdf --parse — 结构化JSON
  • --model / -o / -v — 模型/输出/调试
